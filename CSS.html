Pseudo Class Selectors

There are several states like active, visited, hover, focus(in input for forms), first-child etc..We can use colon(:) for it

Example : 
                    
                        <style>
                                a:visited{
                                    color: rgb(98, 222, 222);
                                }
                                a:active{
                                    color: rgb(29, 221, 29);
                                }
                                a:hover{
                                    color: darkorchid;
                                }
                                input:focus{
                                    color: rgb(238, 149, 76);
                                }
